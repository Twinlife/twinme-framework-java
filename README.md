## Twinme Framework Library

This is the Java library representing more high level and complex operations on top of the [Twinlife Framework](https://github.com/Twinlife/twinlife-framework-java)
library.  It is used by [twinme](https://www.twin.me), [twinme+](https://www.twin.me) and [Skred](https://www.skred.mobi).
It provides:

- Profile representation and management (a Profile allows you to invite relatives and create relations),
- Contact, Group, Click-to-Call representation and management,
- Space management to group the above entities for user's management.
